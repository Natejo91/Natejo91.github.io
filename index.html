Projects

Adopt-a-Pet

Adopt-a-Pet is a clone of the popular PetFinder website which connects users to animals from animal shelters all over the United States. Adopt-a-Pet utilizes 4 different API's to make the website realistic as it is using real animals and animal shelters from PetFinders API. Google Maps API brings everything together as it displays where the animal shelters are located using markers and Info Windows. Favoriting animals is another feature that allows you to favorite animals with a heart icon which then allows you to view them on the favorite animal page. This project utilizes React / Redux, Python, Flask, SQLAlchemy, JavaScript, and CSS.

Bazaar

Bazaar is an online marketplace and a clone of Etsy. Upon coming to the splash page you will see a list of products that are pulled directly from Etsy's API with dynamic seeders. Using real information from Etsy's API gives the website a realistic feel and allows for real information to be displayed on Bazaar. After browsing through products you can add products to your cart and eventually check out and complete your order. As a user you are also allow to create your own store so that you can upload products to the store as an owner. Amazon Web Services is used to allow users to upload product photos and store photos. This project utilizes React / Redux, Python, Flask, SQLAlchemy, JavaScript, and CSS.

GitGud

The app is a question-and-answer website for all things gaming. Users can post questions for other users to respond to, share tips and tricks, and discuss topics related to gaming. Users can register and login or use the demo account. When logged in, users can search for anything in the search bar, search for posts associated with pre-defined topics, as well as create a post or comment. Users can also edit their profile information. This project utilizes JavaScript, Express, Sequelize, Pug, PostgreSQL, HTML, and CSS.

AirStage

AirStage is a website that allows musicians to pick and choose different homes to perform concerts in depending on location. Musicians should be able to view a list of Venues while on the home page. The website will also use a google api to determine where the musician is located and populate the list of venues accordingly. There will also be a search function that will allow musicians to search for a venue depending on location. The venues themselves will have a picture and details about the space and will also have reviews on them by other musicians. This project utilizes JavaScript, React / Redux, Express, Sequelize, PostgreSQL, HTML, and CSS.
