Adopt-a-Pet

Adopt-a-Pet is a clone of the popular PetFinder website which connects users to animals from animal shelters all over the United States. Adopt-a-Pet utilizes 4 different API's to make the website realistic as it is using real animals and animal shelters from PetFinders API. Google Maps API brings everything together as it displays where the animal shelters are located using markers and Info Windows. Favoriting animals is another feature that allows you to favorite animals with a heart icon which allows you to view them then on the favorite animal page. This project utilizes React / Redux, Python, Flask, SQLAlchemy, JavaScript, and CSS.

GitGud

The app is a question-and-answer website for all things gaming. Users can post questions
for other users to respond to, share tips and tricks, and discuss topics related to gaming.
Users can register and login or use the demo account. When logged in, users can search for anything in the search bar, search for posts associated with pre-defined topics,
as well as create a post or comment. Users can also edit their profile information.
This project utilizes JavaScript, Express, Sequelize, Pug, HTML, and CSS.

AirStage

AirStage is a website that allows musicians to pick and choose different homes to perform
concerts in depending on location. Musicians should be able to view a list of Venues while
on the home page. The website will also use a google api to determine where the musician is located and populate the list of venues accordingly. There will also be a search function
that will allow musicians to search for a venue depending on location. The venues themselves
will have a picture and details about the space and will also have reviews on them by other musicians. This project utilizes JavaScript, React / Redux, Express, Sequelize, HTML, and CSS.
